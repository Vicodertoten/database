# INT-019 - Formalisation owner-side des operations editoriales critiques

## Status

- closed
- date: 2026-04-19

## Scope

Formaliser un transport owner-side write minimal pour les operations editoriales critiques,
separe du service runtime-read et borne au perimetre pack/enrichment.

- ADR dediee write transport v1
- schemas operation envelopes versionnes
- service HTTP write minimal avec erreurs categorisees
- reutilisation de `pack_store` et `enrichment_store` existants (pas de duplication metier)
