# Archive Index

This directory stores historical documentation and evidence. Archived documents
are preserved for context only and must not be used as current implementation
guidance.

Current contract source of truth: `../architecture/contract-map.md`.

- `superseded-contracts/`: retired runtime-read, materialization, and contract-transition plans.
- `runbooks/`: closed execution runbooks.
- `chantiers/`: closed inter-repo chantier records.
- `integration-logs/`: monthly archived integration logs.
- `evidence/`: dated campaign artifacts and reports.
- `audits/`: dated audit snapshots.
